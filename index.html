Transformaciones CSS

¿Qué es transform?

Transform es un módulo de CSS que permite manipular visualmente elementos HTML en un espacio bidimensional o tridimensional. Podemos inclinar, rotar, escalar y trasladar elementos sin afectar el flujo normal del documento.

La sintaxis básica es: transform: función(valor);

Scale (Escalar)

La función scale() aumenta o disminuye el tamaño de un elemento. Podemos escalar en el eje X, en el eje Y, o en ambos.

transform: scaleX(1.5); /* Solo eje X */ transform: scaleY(1.5); /* Solo eje Y */ transform: scale(1.5); /* Ambos ejes */ transform: scale(1.5, 1.2); /* Valores diferentes para X e Y */

Skew (Sesgar)

La función skew() inclina un elemento en los ejes X y/o Y. El valor se especifica en grados (deg).

transform: skewX(20deg); /* Solo eje X */ transform: skewY(20deg); /* Solo eje Y */ transform: skew(20deg, 10deg); /* Ambos ejes */

Rotate (Rotar)

La función rotate() gira un elemento alrededor de un punto fijo. El valor se especifica en grados (deg).

transform: rotate(45deg); /* Rotación en sentido horario */ transform: rotate(-45deg); /* Rotación en sentido antihorario */

Sin rotación

Rotado 45 grados

Translate (Trasladar)

La función translate() mueve un elemento desde su posición original. Podemos mover en el eje X, en el eje Y, o en ambos.

transform: translateX(20px); /* Solo eje X */ transform: translateY(20px); /* Solo eje Y */ transform: translate(20px, 20px); /* Ambos ejes */

Transform Origin (Origen de transformación)

La propiedad transform-origin define el punto desde donde se aplica la transformación. Por defecto es el centro del elemento.

transform-origin: top left; /* Esquina superior izquierda */ transform-origin: center; /* Centro (valor por defecto) */ transform-origin: bottom right; /* Esquina inferior derecha */ transform-origin: 20px 50px; /* Coordenadas específicas */

Transformaciones combinadas

Podemos combinar varias transformaciones en una sola declaración. El orden importa: las transformaciones se aplican de derecha a izquierda.

transform: rotate(15deg) scale(1.2) translateX(10px); /* Primero traslada, luego escala y finalmente rota */
