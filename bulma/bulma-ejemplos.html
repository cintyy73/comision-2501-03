¿Qué es Bulma?

Bulma es un framework CSS moderno basado en Flexbox que permite crear interfaces responsivas sin necesidad de JavaScript adicional. Este ejemplo muestra varios componentes usando solo clases de Bulma (sin CSS personalizado).

Ventajas de Bulma:

  • Solo CSS (no requiere JavaScript)
  • Basado en Flexbox
  • 100% responsive
  • Clases intuitivas y legibles
