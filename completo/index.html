Demo API — Clase 13

CORS • Fetch • then/catch • async/await • CRUD

Guía rápida (teoría resumida)

  • CORS es una política de seguridad del navegador: solo el servidor puede habilitar el acceso entre orígenes distintos.
  • fetch hace peticiones HTTP asíncronas; devuelve una Promise.
  • .then() maneja la resolución exitosa; .catch() maneja errores.
  • async/await es azúcar sintáctico sobre promesas, con try/catch para errores.
  • CRUD: GET (leer), POST (crear), PUT (editar), DELETE (borrar).
